Bushalte API

Bezig met laden van XML data...

Documentatie

Deze API laat je zoeken naar bushaltes op basis van (een deel van) de naam. De resultaten bevatten de quaycode, straat en plaatsnaam van overeenkomende haltes.

Gebruik

Gebruik de URL parameter naam om te zoeken naar een bushalte (minimaal 3 letters):

https://jouw-gebruikersnaam.github.io/bushalte-api/?naam=flo

Parameters

Parameter Beschrijving Verplicht
naam De (gedeeltelijke) naam van de bushalte (min. 3 letters) Ja
format Gewenste responsformaat: 'json' (standaard) of 'html' Nee

Voorbeelden

Zoek 'flo' (HTML) Zoek 'lei' (HTML) Zoek 'kap' (HTML) Zoek 'hid' (HTML)

Response formaat (JSON)

{
  "success": true,
  "query": "zoekterm",
  "count": 1,
  "results": [
    {
      "name": "Naam van de halte",
      "quaycode": "NL:Q:12345678",
      "street": "Straatnaam",
      "town": "Plaatsnaam"
    }
  ]
}
